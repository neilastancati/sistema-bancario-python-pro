# 🏦 Sistema Bancário Digital Pro

Este projeto é uma evolução robusta do desafio de sistema bancário, transformando um script simples em uma **API RESTful** com interface gráfica e persistência de dados real.

---

## 🛠️ Tecnologias e Conceitos Aplicados

- **Python & Flask:** Arquitetura de API para transações bancárias.
- **Persistência JSON:** Banco de dados em formato JSON para garantir que os saldos não se percam.
- **UI Moderna:** Protótipo de interface desktop desenvolvido com `CustomTkinter`.
- **Arquitetura Limpa:** Separação total entre lógica de negócio, API e interface.
- **Desenvolvimento Mobile:** Projeto desenvolvido inteiramente em ambiente mobile, demonstrando agilidade e domínio de ferramentas de versionamento.

---

## 🚀 Funcionalidades

- [x] Criar novo usuário com validação de CPF.
- [x] Depósitos e Saques com atualização em tempo real no banco de dados.
- [x] Consulta de extrato via API.
- [x] Persistência total dos dados em arquivo `.json`.
